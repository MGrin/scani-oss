# `@scani/docs`

The Scani OSS documentation site. [Astro](https://astro.build) +
[Starlight](https://starlight.astro.build).

## Local dev

From the repo root:

```bash
bun run dev:docs        # → http://localhost:4321/scani-oss
```

Or directly:

```bash
bun --cwd apps/frontend/docs dev
```

## Build

```bash
bun --cwd apps/frontend/docs build       # → apps/frontend/docs/dist/
bun --cwd apps/frontend/docs preview     # serve the built output locally
```

## Authoring

Content lives in [`src/content/docs/`](./src/content/docs/) as Markdown /
MDX. The sidebar is configured in [`astro.config.mjs`](./astro.config.mjs).

Adding a new page: drop a `.md` file under `src/content/docs/<section>/`,
then register it in the `sidebar` entry for that section. The
`reference/` section is autogenerated — new pages there appear without
config changes.

## Deploy

Static output (`dist/`) deploys to any static host. The site is configured
for GitHub Pages with `base: /scani-oss` in `astro.config.mjs`. Change
`site` / `base` if hosting elsewhere.
